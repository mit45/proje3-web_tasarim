# proje3-web_tasarim

Basit bir HTML/CSS/vanilla JS proje tasarımı. İçerik Firebase Auth ve Firestore ile entegre edilmiştir.

Hızlı başlangıç:

- Tarayıcıda `index.html` dosyasını açarak yerel inceleyebilirsiniz.
- Firebase yapılandırması `assets/js/config/firebaseConfig.js` içinde yer alır.

GitHub Pages (varsayılan): `gh-pages` branch'i oluşturuldu — ayarlar kısmından aktif edilirse site otomatik yayınlanır.
